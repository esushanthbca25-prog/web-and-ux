Two-Column Layout Using Floats

Left Column (Float)

This column floats to the left and takes about 48% of the container's width.

Floats require clearfix or overflow hidden on the container to avoid collapsing.

Right Column (Float)

This column also floats left but has no right margin, completing the row.

Floats were widely used before flexbox and grid became popular.

Two-Column Layout Using CSS Grid

Left Column (Grid)

This column uses CSS Grid and takes half of the available space.

Grid makes layouts more flexible and easier to maintain.

Right Column (Grid)

The gap property adds space between the columns automatically.

CSS Grid is great for complex layouts but works well for simple ones too!
